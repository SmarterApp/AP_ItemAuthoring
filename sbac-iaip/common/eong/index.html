Documentation
readme.htm The readme file in HTML format. Contains late-breaking release notes about system requirements, installation, integration, release history, etc.
Integration Manual - HTML Developer manual in HTML format (best suited for viewing on screen). It contains information about the integration of edit-on NG in a web page and about the more advanced integration possibilities. Also contains the edit-on NG API reference.
Integration Manual - PDF Developer manual in PDF format (best suited for printing). It contains information about the integration of edit-on NG in a web page and about the more advanced integration possibilities. Also contains the edit-on NG API reference.
Samples
Wiki-like article with style templates Demonstrates how you can create a Wiki-like article using style templates.
Form Letter Demonstrates the usage of read-only parts and placeholders based on a form letter.
In place editing with detached User Interface Demonstrates in place editing within a website. The editor has the same size as the content to be edited, the tool and status bar float above / below without influencing the layout of the website.
Technical documentation with style templates Demonstrates how you can create a technical documentation like the edit-on NG "Integration Manual" from scratch using style templates and customizations in the editor.
Minimalistic Editor Demonstrates the possibility to use edit-on NG in a minimalistic way. The status bar is disabled and the tool bar contains no tab bar and panel descriptions.
XPath Demonstrates how to use XPath to move the caret, select elements or manipulate the attributes of elements.
JavaScript API Demonstrates parts of the powerful JavaScript API of edit-on NG.
Multi language and BIDI Demonstrates the usage of multiple languages in one document / support for BIDI.
SVG and MathML Demonstrates the rendering capabilities of edit-on NG for the ML representation of SVG and MathML.
Multiple Instances Demonstrates the integration of multiple edit-on NG instances. In one instance the heading of the article can be edited, the other edits the article itself. In both instances the documentcounter is enabled.
Custom dialogs Demonstrates the integration of custom Java and JavaScript dialogs.
